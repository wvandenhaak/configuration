Configuration file
==================

An example of the contents of the file when a Config is saved to disk.

```php
return [
    'configuration' => [
        // String
        'key_1' => 'a',
        
        // Array
        'key_2' => [
            'value_2_1',
            'value_2_2',
            'value_2_3',
            'value_2_4'
        ],
        
        // Boolean
        'key_3' => false,
        
        // Integer
        'key_4' => 438,
    ]
];
```
